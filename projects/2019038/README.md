<h2><span id="ΕΙΣΑΓΩΓΗ">ΕΙΣΑΓΩΓΗ</span></h2>

Μέσω του μαθήματος της Τεχνολογίας Λογισμικού αποσκοπώ στο να εξασκήσω και να μάθω πρακτικές για τη σχεδίαση και την ανάπτυξη λογισμικού, με τη βοήθεια και την 
ολοκλήρωση των εβδομαδιαίων προγραμματιστικών ασκήσεων και γραμμής εντολών. Επιπλέον, θα μάθω να συνεργάζομαι και με άλλα άτομα με τη δημιουργία ομάδας, μιας και θα 
μπορούμε να ανταλλάσσουμε ιδέες και γνώσεις, κάτι που το θεωρώ πολύ σημαντικό στο χώρο της πληροφορικής. Τέλος, ευελπιστώ να καταφέρω να αναπτύξω το δικό μου προσωπικό 
έργο και να γίνω ευέλικτη στη δημιουργία και συντήρηση εφαρμογών λογισμικού.
